# Статус Arcane Code 1.0.0-dev

Это development-сборка этапа A, а не release candidate и не стабильный релиз 1.0.0.

Проверено:

- Gradle Wrapper 8.10.2 и Java 21;
- `clean build` без ошибок компиляции;
- основной JAR `Arcane-Code-NeoForge-1.21.1-1.0.0-dev.jar`;
- содержимое JAR, JSON, локализации, модели и текстуры;
- условная загрузка рецептов Create;
- dedicated server без Create до состояния `Done`;
- отсутствие `ERROR` и исключений в логе повторного серверного запуска.

Не проверено или не реализовано:

- клиентский запуск, создание мира и подключение к серверу;
- реальная работа всех рун и интерфейсов;
- unit-тесты и GameTests;
- запуск с Create 6.x;
- безопасность payload и мировых операций для публичного сервера;
- полный Кодекс, обучение, конфигурация и команды администратора;
- все критерии готовности стабильной версии 1.0.0.

Подробности: `docs/CURRENT_STATE_AUDIT_RU.md`.
