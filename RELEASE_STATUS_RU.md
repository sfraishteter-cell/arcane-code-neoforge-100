# Статус Arcane Code 1.0.0

Это **release candidate**, а не подтверждённый финальный релиз.

Подготовлено:
- геометрический редактор рун;
- типизированный стек и интерпретатор;
- 142 базовые руны и динамические руны реестров;
- циклы и цитирование программ;
- эффекты, визуальные фигуры и временные физические барьеры;
- Арканная пыль и Резонансный жернов;
- четыре уровня посохов, носители программ и рецепты;
- интерактивный Кодекс;
- data-driven рецепты Create;
- оригинальные модели, текстуры и локализации;
- GitHub Actions и проверка содержимого JAR.

Ещё требуется:
1. успешная настоящая Gradle-сборка;
2. исправление всех ошибок компиляции по логу Actions;
3. тест клиента и dedicated server;
4. проверка совместимости с актуальным Create 6.x;
5. балансировка стоимости рун, медиа и пределов операций;
6. игровые тесты интерфейса, барьеров и циклов.

Версию нельзя честно называть «лучше Hex Casting» до прохождения этих проверок и длительного игрового тестирования.
