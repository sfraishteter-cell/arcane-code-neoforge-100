# Arcane Code 1.0.0 — Release Candidate

- Rebuilt spell creation around geometric rune drawing.
- Added 142 core runes and dynamic registry-backed runes.
- Added typed iotas, quoted programs, conditionals and six loop/data-processing operations.
- Added all-registry effect handling, particle figures, entity/world actions and temporary physical barriers.
- Added Iron Dust, Amethyst Dust and Arcane Dust as the media economy.
- Added the durable Resonance Mill for manually producing both source dusts.
- Added Spark, Resonance, Architect and Sovereign staff tiers with distinct capacity and efficiency.
- Added Rune Scroll, Focus Prism and Woven Artifact program carriers.
- Added an interactive Arcane Codex with chapters, navigation, search and live rune catalog.
- Replaced placeholder art with original item/block textures, translated names and complete recipes.
- Added optional Create 6.x Milling, Crushing and Mechanical Mixing recipes.
- Added GitHub Actions production build and release-JAR structure verification.
