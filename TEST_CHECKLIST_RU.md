# Arcane Code 1.0.0 — тест-лист релиза

## Загрузка
- [ ] Minecraft 1.21.1, NeoForge 21.1.238+, Java 21.
- [ ] В `mods` только один Arcane Code JAR.
- [ ] Главное меню открывается без окна `failed to load correctly`.
- [ ] Мод отображается в списке модов как версия 1.0.0.

## Ресурсная цепочка
- [ ] Резонансный жернов создаётся по рецепту.
- [ ] Железный слиток во второй руке превращается в 2 Железные пыли.
- [ ] Осколок аметиста превращается в 2 Аметистовые пыли.
- [ ] Жернов теряет прочность, но не расходуется за одно применение.
- [ ] Железная + Аметистовая пыль дают 2 Арканные пыли.

## Кодекс
- [ ] Кодекс автоматически выдаётся один раз.
- [ ] Работают главы, назад/вперёд, главная, поиск и прокрутка каталога.
- [ ] Отображаются базовые и динамические руны.

## Посохи и магия
- [ ] Все четыре посоха имеют правильные названия и текстуры.
- [ ] ПКМ открывает сетку рисования.
- [ ] Поворот руны распознаётся одинаково.
- [ ] Отмена, очистка, сохранение и запуск работают.
- [ ] Арканная пыль автоматически пополняет медиа посоха.
- [ ] Сбои стека, типа, медиа и лимита отображаются без падения игры.
- [ ] Циклы Repeat, For Each, Map, Filter, Fold и While не зависают.
- [ ] Эффекты, частицы и временные барьеры работают на сервере.

## Create
- [ ] Без Create мод запускается.
- [ ] С Create 6.x мод запускается.
- [ ] Millstone/Crushing Wheels создают обе пыли.
- [ ] Mechanical Mixer создаёт Арканную пыль.

## Финал
- [ ] Клиент закрывается без crash report.
- [ ] Проверен отдельный NeoForge-сервер.
- [ ] Клиент успешно подключается к серверу с тем же JAR.
