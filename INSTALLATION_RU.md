# Arcane Code 1.0.0 — установка и проверка

## Требования
- Minecraft Java 1.21.1
- NeoForge 21.1.238 или новее в ветке 21.1.x
- Java 21
- Create 6.x необязателен: без него мод работает самостоятельно

## Установка в CurseForge
1. Закрой Minecraft.
2. Открой профиль `hex 2.0` в CurseForge.
3. Нажми три точки → **Open Folder**.
4. Открой папку `mods`.
5. Удали все прежние `Arcane-Code*.jar`.
6. Положи туда `Arcane-Code-NeoForge-1.21.1-1.0.0.jar`.
7. Запусти профиль.

## Первые действия
1. На первом входе игрок автоматически получает **Арканный Кодекс**.
2. Создай **Резонансный жернов**.
3. Возьми жернов в одну руку, железный слиток или осколок аметиста — в другую.
4. Нажми ПКМ жерновом: он создаст соответствующую пыль и потеряет 1 единицу прочности.
5. Смешай Железную и Аметистовую пыль, чтобы получить Арканную пыль.
6. Создай посох, открой его ПКМ, рисуй руны на шестиугольной сетке.
7. Shift + ПКМ посохом запускает сохранённое плетение.

## Create
При установленном Create доступны автоматические рецепты:
- Millstone и Crushing Wheels производят обе исходные пыли;
- Mechanical Mixer смешивает их в Арканную пыль.
