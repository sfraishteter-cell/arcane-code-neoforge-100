# Arcane Code 1.0.0 — каталог базовых рун

В этой версии определено **142 базовых рун**. Дополнительно при запуске автоматически создаются руны для каждого эффекта, простой частицы, блока, предмета, звука и типа сущности, зарегистрированных в Minecraft и установленных модах.

Геометрия каждой руны отображается внутри Арканного Кодекса; поворот фигуры не меняет её значение.

## Динамические руны

- `effect/<namespace:id>` — кладёт зарегистрированный эффект на стек.
- `particle/<namespace:id>` — кладёт простую частицу на стек.
- `block/<namespace:id>` — кладёт блок на стек.
- `item/<namespace:id>` — кладёт предмет на стек.
- `sound/<namespace:id>` — кладёт звук на стек.
- `entity_type/<namespace:id>` — кладёт тип сущности на стек.

Количество динамических рун зависит от состава модпака; Кодекс строит каталог при запуске.
