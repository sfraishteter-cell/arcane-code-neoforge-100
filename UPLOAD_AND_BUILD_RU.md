# Arcane Code 1.0.0 — загрузка в GitHub и получение JAR

## Почему в архиве проект, а не готовый мод

Готовый NeoForge-мод создаётся Gradle-сборкой с настоящими библиотеками Minecraft и NeoForge. Встроенный GitHub Actions делает такую сборку на Java 21 и проверяет структуру JAR. Не переименовывай ZIP в JAR.

## 1. Создай чистый репозиторий

1. Открой GitHub и нажми `+` → **New repository**.
2. Название: `arcane-code-neoforge-100`.
3. Можно выбрать **Private** или **Public**.
4. Не добавляй README, `.gitignore` и лицензию: они уже лежат в проекте.
5. Нажми **Create repository**.

## 2. Загрузи проект

1. Распакуй архив проекта.
2. Открой уровень, где рядом видны `.github`, `src`, `build.gradle` и `gradle.properties`.
3. В пустом репозитории нажми **uploading an existing file**.
4. Выдели всё содержимое распакованной папки сочетанием `Ctrl+A` и перетащи на GitHub.
5. Убедись, что загрузилась папка `.github/workflows`.
6. Сообщение коммита: `Upload Arcane Code 1.0.0 release candidate`.
7. Нажми **Commit changes**.

## 3. Дождись сборки

1. Открой вкладку **Actions**.
2. Выбери **Build Arcane Code 1.0.0**.
3. Жёлтый кружок означает, что сборка идёт.
4. Зелёная галочка означает, что JAR собран и прошёл автоматическую проверку.
5. Красный крест означает ошибку; открой шаг **Build and verify** и скачай/скопируй его лог.

## 4. Скачай мод

1. Открой успешный запуск Actions.
2. Внизу страницы найди **Artifacts**.
3. Скачай `Arcane-Code-NeoForge-1.21.1-1.0.0`.
4. Распакуй скачанный ZIP.
5. Внутри находится `Arcane-Code-NeoForge-1.21.1-1.0.0.jar` — это и есть файл мода.

## 5. Установи в CurseForge

1. Полностью закрой Minecraft.
2. CurseForge → Minecraft → `hex 2.0` → три точки → **Open Folder**.
3. Открой `mods`.
4. Удали старые файлы `Arcane-Code*.jar`.
5. Положи только новый JAR.
6. Для интеграции установи Create 6.x для Minecraft 1.21.1; без Create Arcane Code также должен запускаться.

## 6. Обязательная проверка

- Мод отображается как `Arcane Code 1.0.0`.
- На первом входе выдаётся Арканный Кодекс.
- ПКМ Кодексом открывает главы, поиск и каталог рун.
- Резонансный жернов создаёт по 2 пыли из железного слитка/аметистового осколка во второй руке.
- Две пыли смешиваются в Арканную пыль.
- ПКМ посохом открывает Палату Начертания.
- Сохранение и Shift+ПКМ запускают программу.
- В `latest.log` нет исключений `arcanecode`.
- При установленном Create видны рецепты Milling, Crushing и Mixing.
